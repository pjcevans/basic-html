About Me

I have a growing interest in web development and love creating systems that work and are useful. As a hobby I enjoy learning about new technologies, running through tutorials and building on where they finish.

I am now really keen to take my interest in coding to the next level, giving it the attention it deserves so I can work with others to build effective and beautiful systems.

Below is a breakdown of some of the areas I have explored while building my interest in development.

Python

I have used python to explore the fundamentals of object oriented programming, I enjoy its clean intuitive style and the importance placed on use of whitespace.

Ruby

I've used Ruby to write some basic web scrapers and as part of implementing Rails applications. I would like to work with Ruby in much more depth, both standalone and in conjunction with Rails.

C

I studied C to a basic level in order to learn the fundamentals of programming. I've found other languages more intuitive and engaging since but enjoyed touching on pointers and bitwise operations, which I have yet to do since.

Angular

I completed a tutorial which implemented an Angular front-end on a rails back-end (serving JSON, asset pipeline disabled). I found this to be very enjoyable as I love the idea of effective single page applications

Heroku

I think Heroku is great, I love the way hardware capacity is abstracted into "dynos". I've deployed using Heroku while taking my first steps with Rails.

Rails

I have been interacting with Rails as part of work but have recently been using tutorials to develop a familiarity with Rails. My most recent success has been re-writing migrations to be able to deploy an app where I had manually modified the db during development.

Django

I worked on the tutorials included in the Django documentation when I was first starting to find out about back-end development. Though I didn't progress far with Django I did learn the basics of MVC.

HTML

I first started building websites for fun using just HTML and CSS, I enjoy working with the Document Object Model using HTML.

CSS

I enjoy working with CSS and have enjoyed focussing on optimisation in the past. I've experimented with transitions for the first time in making this site and am looking forward to becoming adept in newer CSS capabilities.

Twitter Bootstrap

Though I've not used it here as I thought it appropriate to work from scratch I've worked through the Bootstrap documentation before as I think relying on a framework for common optimisations and design patterns could be very useful in certain situations

Wordpress

I have built a few websites with Wordpress, the only one currently live is worldeducationalfilms.com - Reaching the extent of what was possible with customising Wordpress templates was what led me to spend more time understanding web back-end.

CLI

I have used the bash command line for a range of tasks such as using rbenv to manage ruby versions, managing file structure, piping output through various tools including jq for processing JSON.
